# TestFlight-Auto-Invite-PHP

A small Python script through which an email can be added into the TestFlight automatically. 

# How To
Just need to pass

1. User email id.
2. Your application ID.
3. User First Name.
4. User Last Name.

and call "testflight_invite.php" file. Thats it.
